## What Was Captured

The `netcap.pcapng` file contains the following network traffic:

### ICMP Traffic (Primary Focus)
- **ICMP Echo Requests (Type 8)** - Ping packets sent from local machine to google.com
- **ICMP Echo Replies (Type 0)** - Response packets from Google's servers
- **Sequence Numbers** - Matching request/reply pairs
- **TTL Values** - Time to Live showing number of network hops
- **Round Trip Time** - Latency measurements between request and reply

### DNS Traffic
- **DNS Query** - Request to resolve google.com domain name to IP address
- **DNS Response** - Answer containing Google's IP address(es)
- **Query Type** - A record (IPv4 address lookup)

### Additional Network Data
- **Ethernet Frames** - Layer 2 headers with MAC addresses
- **IP Headers** - Layer 3 information with source/destination IPs
- **Timestamps** - Precise timing of each packet
- **Packet Lengths** - Size of each captured packet
